🧾 About

Isharah is a large-scale dataset for Continuous Saudi Sign Language (SSL) recognition and translation. It features over 30,000 video samples signed by deaf and hearing-impaired individuals using smartphones in varied settings.

The dataset supports both Continuous Sign Language Recognition (CSLR) and Sign Language Translation (SLT), and includes:

  • Sentence-level gloss annotations
  • Corresponding Arabic translations
  • Three benchmark subsets: Isharah-500, Isharah-1000, and Isharah-2000
